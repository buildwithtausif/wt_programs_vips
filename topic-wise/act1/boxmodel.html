The CSS Box Model

Concept
In CSS, every HTML element is treated as a rectangular box. The "box model" describes how this box is constructed.
The Layers (from inside out)
  1. Content: The actual text, image, or other content.
  2. Padding: The clear space around the content, inside the border.
  3. Border: The line that goes around the padding and content.
  4. Margin: The clear space outside the border, separating this element from other elements.
box-sizing: border-box;
This is a very useful CSS rule (you're using it!). By default, an element's width only applies to the content. border-box makes the width include the content, padding, AND border, which is much more intuitive for layout.
