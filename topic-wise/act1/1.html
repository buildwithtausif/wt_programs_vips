What is HTML?

1.1 Overview
HTML is a markup language, which is used to create basic structure of a webpage.
1.2 Markup language
A markup language is a system for annotating a document with symbols or tags to define its structure, formatting, and the relationship between its parts.
1.3 Generations of HTML

HTML (HyperText Markup Language) has evolved significantly since its inception. Here are its key generations:

  • HTML (Approx. 1991): The initial, informal version created by Tim Berners-Lee to share documents.
  • HTML 2.0 (1995): The first official standard (RFC 1866). It formalized core features like headings, lists, and forms.
  • HTML 3.2 (1997): A W3C standard that added widely-used features from browser vendors, including tables, applets, and text flow around images.
  • HTML 4.01 (1999): A long-lasting and important standard. It strongly emphasized the separation of content structure (HTML) from presentation, pushing the use of CSS (Cascading Style Sheets) for styling.
  • XHTML 1.0 (2000): A reformulation of HTML 4.01 as a stricter XML (Extensible Markup Language) application. It enforced stricter syntax rules (e.g., all tags must be closed).
  • HTML5 (2014): The current major standard. It introduced many new features for modern web applications, including:
    • New semantic elements like <article>, <section>, <nav>, and <header>.
    • Native support for <video> and <audio> elements.
    • New form controls (e.g., date, email, tel).
    • The <canvas> element for 2D drawing and JavaScript APIs.
