HTML Lists

Unordered List <ul>
Used for lists where the order doesn't matter. Items are marked with bullets. Uses <li> (list item) tags inside.
Ordered List <ol>
Used for lists where the order is important (e.g., steps in a recipe). Items are numbered. Also uses <li> tags inside.
Description List <dl>
Used to list terms and their descriptions (like a dictionary). Uses <dt> (description term) and <dd> (description details). The main page content uses this!
