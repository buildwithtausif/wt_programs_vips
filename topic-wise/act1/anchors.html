HTML Anchors (Links)

Anchor Tag <a>
The anchor tag creates a hyperlink, which is a clickable link to another document or resource.
The href Attribute
This is the most important attribute. It specifies the URL (destination) of the link.
<a href="https://www.google.com">Go to Google</a>
The target Attribute
Specifies where to open the link. The value _blank is very common; it opens the link in a new browser tab.
<a href="index.html" target="_blank">Open Home in New Tab</a>
